# Aula 01 – Introdução à Programação Orientada a Objetos

## Conteúdos
- Programação procedural (variáveis e funções)
- Introdução a objetos (atributos e métodos)
- Acesso a propriedades (notação de ponto e colchetes)

## Como executar
No terminal, dentro desta pasta:
```bash
node 01_programacao_procedural.js
node 02_objeto_literal.js
node 03_acesso_atributos.js
```
