# Aula 03 – Funções, Arrays, Módulos e Assíncrono

## Conteúdos
- var, let e const (escopos e comportamento)
- Funções: parâmetros, default, closures
- Arrow functions
- Funções de alta ordem em arrays (some, every, filter, forEach, reduce, map)
- Modularização (CommonJS e ES Modules)
- JSON (stringify/parse) e limitações
- Exceções (try/catch/finally)
- Assíncrono: callbacks, promises e async/await

## Como executar
No terminal, dentro desta pasta:
```bash
node 01_var_let_const.js
node 02_funcoes_e_closure.js
node 03_arrow_functions.js
node 04_arrays_metodos.js
node 05_higher_order_arrays.js
node 06_modules_commonjs.js
node 07_json.js
node 08_excecoes.js
node 09_promises_async_await.js
node 10_callbacks_fs.js
```
