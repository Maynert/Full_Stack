# Programação Orientada a Objetos

Este repositório reúne **exemplos práticos** (em JavaScript/Node.js) baseados nos **exercícios e trechos apresentados nas 3 aulas**, cobrindo:
- **Objetos** (literais, acesso/atribuição, métodos)
- **Classes** (constructor, métodos, herança e polimorfismo)
- **Protótipos**
- **Membros estáticos**
- **Encapsulamento** (campos e métodos privados `#`, getters/setters)
- **Associação entre objetos** (Aluno ↔ Nota)
- **Funções** (closure e arrow functions)
- **Funções de alta ordem** em arrays (`some`, `every`, `filter`, `forEach`, `reduce`, `map`)
- **Modularização** (CommonJS e ES Modules)
- **Exceções** (`throw`, `try/catch`, exceção customizada)
- **Assíncrono** (callback, Promise, async/await)

> **Node recomendado:** 18+ (funciona em versões atuais).  
> Os arquivos são independentes; você pode rodar um por vez.

---

## Estrutura

- `README.md` — visão geral do portfólio
- `exemplos/` — scripts organizados por tema (cada arquivo tem comentários no topo)

## Como executar

Entre na pasta `exemplos/` e rode:

```bash
node 01_objeto_literal.js
node 03_classes_heranca_polimorfismo.js
node 07_array_high_order.js
```

### ES Modules (arquivos `.mjs`)
```bash
node 09_modulos_es6/03a_consumidor_ES6.mjs
```
