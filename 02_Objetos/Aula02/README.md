# Aula 02 – Objetos, Protótipos e Classes

## Conteúdos
- Objetos literais e funções fábrica
- Funções construtoras e uso de `this`
- Protótipos (`prototype`) e herança prototipada
- Classes ES6 (`class`) e herança (`extends` / `super`)
- Atributos estáticos, privacidade com `#`, getters e setters
- Associação entre objetos (composição)

## Como executar
No terminal, dentro desta pasta:
```bash
node 01_funcao_fabrica.js
node 02_funcao_construtora.js
node 03_object_constructor.js
node 04_prototype.js
node 05_classes_es6.js
node 06_heranca_polimorfismo.js
node 07_estaticos.js
node 08_privados_get_set.js
node 09_associacao.js
```
