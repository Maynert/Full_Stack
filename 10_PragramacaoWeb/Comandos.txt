- Primeiro lugar, pensamos na modelagem para mobile

reposnsáveis por isso são os media queries

são declarações em CSS de forma condicional

permitem aplicar estilos de acordo com o dispositivo

no minuto 19:44, apresenta-se 2 formas para realização

em um element, fica dessa forma:

<link rel='stylesheet'
  media="print"
  href="estilos-impressoa.css" />

Dentro de uma declaração CSS

<style>
  @media print {
    .menu {
      display: none;
    }
  }
</style>

no media queris, usa a largura da tela para identificar o dispositivo

no minuto 22:32 mostra sobre a largura
