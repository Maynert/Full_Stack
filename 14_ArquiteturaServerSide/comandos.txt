no package em Script o  "start": "node server2.js", serve para iniciar automaticamente quando rodar o npm start

se instalar uma biblioteca com -s no final, ele adiciona como dependencia no package

se instalar com -g ele transforma global, não adiciona no package mas ta instalado para uso na máquina.

exemplo: npm install nodemon -g | Esse cara ficará global, se colocar -s ficara como dependencia, sempre levando junto ao fonte.

Instalado o nodemon, para que possa salvar o arquivo e atualizar o server de forma automática, sem precisar reiniciar o server.
