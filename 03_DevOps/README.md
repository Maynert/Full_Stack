# Portfólio — Desenvolvimento Full Stack (PUCRS)

Este repositório reúne evidências, artefatos e registros de aprendizagem produzidos ao longo das disciplinas do curso de Desenvolvimento Full Stack (PUCRS).  
A organização está separada por disciplina, com pastas numeradas para facilitar navegação e referência no Memorial da Jornada de Formação.

## Estrutura padrão por disciplina
Cada pasta de disciplina contém:
- `README.md` — resumo do aprendizado + o que foi produzido
- `artefatos/` — códigos, diagramas, scripts e entregáveis
- `evidencias/` — prints e comprovações de execução/resultado
- `referencias/` — links e materiais de apoio (slides, artigos, vídeos)
