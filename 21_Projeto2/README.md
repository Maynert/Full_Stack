# Portfólio — Desenvolvimento Full Stack (PUCRS)

Este repositório reúne evidências, artefatos e registros de aprendizagem produzidos ao longo das disciplinas de Proejto 2

## Método de execução.

1. Buildar no docker o container: docker compose up --build -d
2. Caso precise excluir os container: docker compose down -v 

### Comandos docker

1. Se quiser instalar nova biblioteca no container: docker compose exec api sh
2. Comando para instalação da nova biblioteca: npm install express
3. Comando para sair do container: Exit
